* 此工具可導出當前遊戲中已加載的 BepInEx 插件和 IPA 插件
* 格式為 Json 和 CSV
* 適配 IPALoaderX v1.2 以上版本
* 重新 Enable 後會立即倒出當前加載清單

## 銘謝

此插件和 Madevil 共同開發

## 需求依賴

* **BepInEx v5.0.1**

## 安裝方式

參考壓縮檔結構，將文件放進「BepInEx/plugins/jim60105」資料夾之下
