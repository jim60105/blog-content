年初我獲得了[這一篇文章](@/AI/say-goodbye-to-ai-hallucinations-a-simple-method-to-Improve-the-accuracy-of-your-rag-system/index.md)在 HackerNoon 投稿得獎的獎金 —— USD $1000，有趣的是主辦單位是以加密貨幣 USDT 的方式支付。過往我就有幾次使用加密貨幣的經驗，但至今都是研究性質的操作小筆金額，雖然對冷錢包也是有興趣，但沒有富有到能「買 3000 元的錢包來裝 30 元」。這次收到這筆款項，再加上閱讀了 [高重建的《每天買一百元比特幣——我的定額定投四年戰績實錄》](https://liker.land/zh-Hant/nft/class/likenft1dpegnjnklnh5g66nn06gz64wv0ec6k04f4v5lx69qlv6xlh493eqdq454j)，讓我決定終於該好好的持有一些加密貨幣了。

{{ image(url="IMG_20250206_223328.jpg") }}

Tangem 冷錢包跟其它的硬體錢包有些不同，它是一張 NFC 卡片，而不像其它的錢包要嘛是一個「USB 隨身碟」，要嘛是一台「機器」。噢，應該說是三張卡片，套裝裡有三張卡片做備援。

卡片型式的設計讓 Tangem 可以做到壽命更長，更耐用，同時又能維持真正的「冷」錢包特性——斷網。

{{ image(url="IMG_20250206_223210.jpg") }}

盒子裡除了規格之外還有一張說明書，簡單的引導你下載 App，然後透過 NFC 連接卡片。

{{ image(url="IMG_20250206_223231.jpg") }}

{{ image(url="IMG_20250206_223242.jpg") }}

App 設計的很簡單，使用上並沒有困難的地方。在建立私鑰的過程中會引導你反覆的讀取三張卡片，在設定完成之後會來到這個主畫面。我認為這個 App 的設計很清爽，不像其它的錢包 App 總是充滿各種廣告、推銷、或是花俏的新功能宣傳橫幅。

冷錢包的作用單純，而 App 也應該是如此。

{{ image(url="Screenshot_2025-02-07-19-44-10-439_com.tangem.wallet.jpg") }}

畫面上的三個功能分別是「購買」、「交易」、「銷售」。

點下了購買之後會跳轉到 [mercuryo](https://exchange.mercuryo.io/) 信用卡刷卡入金。這個平台的操作簡單，但是用這種方式入金的手續費較高，一般來說不建議這樣使用。從其它的交易所換得加密貨幣再轉入 Tangem 錢包儲存是更好的方式。

「交易」功能是在不同的加密貨幣之間轉換，這個功能整合了幾個中心化/去中心化交易所和跨鏈橋，非常方便。

例如說在 Polygon 鏈上的 USDC 轉換成 Ethereum 鏈上的 ETH，這種跨鏈操作必須要用到 CEX 或是 Bridge 才能完成，選項中就只會顯示這些選項。
