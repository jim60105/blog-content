好久沒寫開箱文了！

去年我在 HackerNoon 投稿了[一篇和 RAG 相關的技術文章](@/AI/say-goodbye-to-ai-hallucinations-a-simple-method-to-Improve-the-accuracy-of-your-rag-system/index.md)，有幸獲選得獎，在農歷年假期間領到了一筆不小的獎金 —— USD $1000。

有趣的是，主辦單位是以加密貨幣 USDT 支付！這是一個跨國競賽，主辦單位選擇以此方式支付獎金可以省下大筆的跨國電匯手續費，我認為是一個很聰明的選擇。

過往我曾有一些使用加密貨幣的經驗，但都是研究性質操作小筆金額。雖然對冷錢包一直都有些興趣，但還沒到「花 3000 元買錢包來存 30 元」的程度。

這次藉由收到這筆款項的機會，再加上閱讀了 [高重建的《每天買一百元比特幣——我的定額定投四年戰績實錄》](https://liker.land/zh-Hant/nft/class/likenft1dpegnjnklnh5g66nn06gz64wv0ec6k04f4v5lx69qlv6xlh493eqdq454j)，讓我決定終於該好好的開始持有一些加密貨幣了。

{{ image(url="IMG_20250206_223328.jpg") }}

Tangem 冷錢包跟其它的硬體錢包有些不同，它是一張 NFC 卡片 —— 噢，應該說是三張卡片，並不像其它的錢包要嘛是一個「USB 隨身碟」，要嘛是一台「機器」。

與 USB 相比，卡片形式使 Tangem 的私鑰實現真正的「冷存儲」，且卡片壽命更長、更耐用。

{{ image(url="IMG_20250206_223210.jpg") }}

盒子裡除了規格之外還有一張說明書，簡單的引導你下載 App，然後透過 NFC 連接卡片。

{{ image(url="IMG_20250206_223231.jpg") }}

{{ image(url="IMG_20250206_223242.jpg") }}

App 設計的很簡單，使用上並沒有困難的地方。在建立私鑰的過程中會引導你反覆的讀取三張卡片，在設定完成之後會來到這個主畫面。我認為這個 App 的設計很清爽，不像其它的錢包 App 總是充滿各種廣告、推銷、或是花俏的新功能宣傳橫幅。

冷錢包的作用單純，而 App 也應該是如此。

{{ image(url="Screenshot_2025-02-07-19-44-10-439_com.tangem.wallet.jpg") }}

畫面上的三個功能分別是「購買」、「交易」、「銷售」。

點下了購買之後會跳轉到 [mercuryo](https://exchange.mercuryo.io/) 信用卡刷卡入金。

用這個方式入金真的是方便簡單，但缺點是手續費較高。從其它的交易所換得加密貨幣再轉入 Tangem 錢包儲存會是划算的方式。

「交易」功能是在不同的加密貨幣之間轉換，它功能整合了數個中心化/去中心化交易所和跨鏈橋，非常方便。

例如說在 Polygon 鏈上的 USDC 轉換成 Ethereum 鏈上的 ETH，這種跨鏈操作必須要用到 CEX 或是 Bridge 才能完成，選項中就只會顯示這些選項。
